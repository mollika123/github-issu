1️⃣ What is the difference between var, let, and const?

answer:
var → Function-scoped, can be re-declared & updated, hoisted.

let → Block-scoped, can be updated but not re-declared, hoisted in TDZ.

const → Block-scoped, cannot be re-assigned, hoisted in TDZ; objects can have mutable properties.


2️⃣ What is the spread operator (...)?

answer:The spread operator (...) in JavaScript is used to expand iterable values (like arrays or objects) into individual elements or properties.



3️⃣ What is the difference between map
(), filter(), and forEach()?


answer:map + filter are functional, chainable, pure → ideal for transformations.

forEach is imperative → ideal for side-effects like DOM updates, console logs, or API calls.

4️⃣ What is an arrow function?

answer:An arrow function is a shorter syntax for writing functions using =>.



5️⃣ What are template literals?

answer:Template literals are strings enclosed by backticks ` instead of single ' or double " quotes.

They allow interpolation (embedding variables or expressions) and multi-line strings.
